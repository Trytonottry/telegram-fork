# telegram-fork
## Добавлена вкладка "Музыка" с поиском и мини-плеером аудио
Добавлена новая вкладка "Музыка" в BottomNavigationView, которая отображает все аудиозаписи из чатов. 
Функционал включает:
- Поиск по названию трека
- Мини-плеер с кнопками Play/Pause/Next/Previous
- Отображение текущего трека

Планы на будущие улучшения (не включены в этот PR):
- Индикатор прогресса трека
- Кэширование для оффлайн воспроизведения
- Сортировка по чату или дате
- UI улучшения и обложки аудио
